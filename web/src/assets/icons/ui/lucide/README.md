Style conventions:
- use Lucide outline icons only; do not mix icon families in the same UI area
- prefer module imports from `web/src/assets/icons/ui/lucide/*.svg`
- prefer `currentColor` + CSS mask rendering for consistent hover/disabled states
- icon is assistive only; keep localized text labels visible

Downloaded for current UI areas:
- top floating controls
- system menu tabs
- status bar entries
- settings panel
- save/load panel
- about / other menu cards

- `menu.svg`: game menu button
